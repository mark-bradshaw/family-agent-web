Privacy Policy

Last Updated: February 10, 2026

Introduction

This Privacy Policy describes how the Bradshaw Family Agent ("we," "our," or "the Agent") handles information when integrating with Google Workspace services. The Bradshaw Family Agent is a custom-built AI assistant designed exclusively for personal use by the Bradshaw family.

Scope and Purpose

The Bradshaw Family Agent is a private, family-only application that runs on a local home server. It is not a public service and is not available to anyone outside the Bradshaw family.

Google API Services

The Agent integrates with Google Workspace services to provide household management features. This integration requires authentication with Google's OAuth 2.0 system.

Data Access and Usage

When authorized, the Agent may access:

  • Google Calendar data for scheduling and event management
  • Gmail for email notifications and management
  • Google Drive for document access and organization
  • Other Google Workspace services as needed for family household operations

All data accessed by the Agent is:

  • Used solely for the purpose of providing household management assistance to the Bradshaw family
  • Processed locally on the family's home server
  • Never shared with third parties
  • Never sold or used for advertising
  • Never stored outside of the local home server environment

Data Storage and Security

The Agent operates on a local home server within the Bradshaw family's private network. All data processing occurs locally, and no user data is transmitted to external servers or cloud services except as required to interact with authorized Google Workspace services.

Data Retention

The Agent may temporarily cache data from Google services to provide responsive assistance. This cached data is stored only on the local home server and is automatically managed according to the application's operational needs.

User Control and Access

Family members can:

  • Revoke the Agent's access to their Google account at any time through their Google Account settings
  • Request deletion of any cached data by contacting the administrator
  • View what permissions the Agent has been granted through Google's security settings

Third-Party Services

The Agent uses the following third-party services:

  • Google Workspace: For calendar, email, and document management features
  • Discord: As the interface for family interaction with the Agent

Each of these services has its own privacy policy that governs how they handle data.

Changes to This Policy

This Privacy Policy may be updated from time to time to reflect changes in the Agent's functionality or legal requirements. The "Last Updated" date at the top of this policy will be revised accordingly.

Compliance

The Bradshaw Family Agent complies with Google's API Services User Data Policy, including the Limited Use requirements. This means that the Agent's use of information received from Google APIs adheres to the following principles:

  • Use of Google user data is limited to providing or improving user-facing features
  • No transfer of Google user data to third parties
  • No use of Google user data for advertising
  • No use of Google user data for purposes unrelated to the item's core functionality
